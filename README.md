## RESUMO

### S.I.G.A - Sistema integrado de gerenciamento de acesso

É imprescindível que uma instituição de ensino que se preze tenha uma maneira prática de identificar os acessos, verificar autorizações de entrada e saída e registrar essas movimentações em seu campus de maneira a centralizar os dados conseguidos e propor uma gestão mais ágil e fácil. Neste contexto, este trabalho tem como objetivo apresentar o sistema S.I.G.A como software de solução tecnológica para realizar o controle de acesso de pessoas ao campus da instituição. Para fazer deste trabalho possível, foram utilizados conceitos lecionados em matérias como a de engenharia de software, tais quais regras de negócio, diagrama de classes para definir suas funções e diversos outros que vieram a contemplar na conclusão do projeto. Além disso, foram utilizados diversos conhecimentos referentes ao desenvolvimento web. Como resultado desta abordagem, identificou-se que o controle de acesso ao campus facilitou a identificação dos servidores, alunos e visitantes presentes contribuindo para uma maior organização por parte da instituição. Como conclusão, registra-se:  i) maior segurança; ii) automatização do gerenciamento de acesso; iii) simplicidade ao acesso de histórico de entrada e saída.

Em conclusão, o sistema SIGA poderá se mostrar uma solução eficaz para  o controle de acesso de pessoas ao campus. A implementação deste software poderá aprimorar a segurança, simplificar o acesso ao histórico de entrada e proporcionar uma gestão mais fácil, trazendo benefícios significativos à instituição.
